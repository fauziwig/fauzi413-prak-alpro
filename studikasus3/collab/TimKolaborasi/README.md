# Perintah Dasar Terminal
### Masuk dan Keluar folder
Masuk Folder
```sh
cd nama_foler
```
Keluar Folder
```sh
cd ..
```

### Membersihkan Layar
```sh
clear
```
### Compile dan Run File C++
compile
```sh
g++ -o outName NameFile.cpp
```
run
```sh
./outName
```
